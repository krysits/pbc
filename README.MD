## PHP Blockchain ##

---

PHP based blockchain with `Proof of Work` pattern used.

---

Requirements: 
- PHP7.4

---
Installation:

`composer require krysits/pbc`

---

- Run tests:

`php tests/example.php`

- or try to validate this blockchain:

`php tests/validate.php`
